# Node-Next
# Node.js - практики в Академии:
# Module 1 - introduction to Node.js.
# Практика
Задание 1.1:
• Установите Node.js через nvm, проверьте версию
• Создайте скрипт info.js, который выводит: дату, платформу, версию
Node, аргументы командной строки, использование памяти
• Поэкспериментируйте с REPL: попробуйте работу с массивами,
объектами, Math, Date
• Напишите пример с setTimeout, setImmediate и Promise —
предскажите порядок вывода, затем проверьте
• Используйте console.time/timeEnd для измерения времени цикла из 1
000 000 итераций

Задание 2.1:
• Создайте проект с npm init, добавьте "type": "module"

проект

затем попробуйте разобраться в описании модуля самостоятельно и
выведете цветной текст в консоль
